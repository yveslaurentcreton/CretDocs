## Install and configure `sudo`

<Steps>

1. Login as `root`.

1. Install `sudo` using:

    ```bash
    apt install sudo
    ```

1. (optional) To check who is allowed to use `sudo`, run:

    ```bash
    cat /etc/sudoers
    ```

1. To add a user to the `sudo` group:

    ```bash
    usermod -aG sudo <username>
    ```

</Steps>


## Import GitHub SSH Keys onto Linux Server

<Steps>

1. Install `ssh-import-id` using:

    ```bash
    sudo apt install ssh-import-id
    ```

1. Import your GitHub SSH keys using:

    ```bash
    ssh-import-id-gh <github-username>
    ```

</Steps>

## Install `curl`

Install `curl` using:

```bash
sudo apt install curl
```

## Configure auto reboot

Add a line to `/etc/crontab` to reboot the system at 3:00 AM every day using:

```bash
echo "0 3 * * * root /sbin/shutdown -r now" | sudo tee -a /etc/crontab
```
